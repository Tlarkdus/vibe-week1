# hello

빌드 도구 없이 브라우저에서 바로 실행되는 간단한 정적 웹페이지입니다.

## 구성

- `index.html` — 페이지 구조(마크업)
- `styles.css` — 스타일(다크 테마)
- `script.js` — 이름 입력 후 인사말을 표시하는 상호작용

## 실행 방법

별도의 빌드나 설치 과정이 없습니다.

- **가장 간단한 방법**: `index.html` 파일을 웹 브라우저로 엽니다.
- **로컬 서버로 실행**(권장, 자동 새로고침·경로 문제 방지):

  ```powershell
  # Python이 설치되어 있다면
  python -m http.server 8000
  ```

  이후 브라우저에서 http://localhost:8000 에 접속합니다.
