# 바이브 코딩 · vibe-week1 🍷

바이브 코딩 실습 결과물을 한 저장소에 모은 프로젝트입니다.
백엔드·빌드 도구·외부 라이브러리 없이 **브라우저에서 바로 열리는 정적 웹페이지**들로 구성했습니다.

동덕여자대학교 대표색인 **와인/자주** 계열을 테마로 사용합니다.

---

## 📂 구성

| 폴더/파일 | 설명 | 바로 보기 |
|-----------|------|-----------|
| [`index.html`](./index.html) | 학생의 출결·성적·과제를 한 화면에서 확인하는 **동덕여대 스타일 LMS 대시보드** (벤토 카드 그리드 + 도넛 차트) | 루트에서 실행 |
| [`hello/`](./hello) | 이름을 입력하면 인사해 주는 **Hello 인사 페이지**. 아날로그 시계, 별밭·별똥별·컨페티 효과 포함 | `hello/index.html` |
| [`guestbook.html`](./guestbook.html) | 이름과 한마디를 남기는 **방명록 페이지**. `localStorage`에 저장되어 새로고침해도 유지, 글 삭제 가능 | 루트에서 실행 |

> 실습이 진행되면서 페이지가 계속 추가됩니다.

---

## 🚀 실행 방법

빌드나 설치 과정이 전혀 없습니다. 두 가지 방법 중 하나로 실행하세요.

### 1) 파일을 브라우저로 바로 열기
- `index.html`(대시보드) 또는 `hello/index.html`(인사 페이지)을 더블클릭

### 2) 로컬 서버로 실행 (권장)

```bash
# Python 내장 서버
python -m http.server 8000

# 또는 Node.js
npx http-server -p 8000
```

접속: http://localhost:8000

---

## 🎨 특징

- **바닐라 스택**: 프레임워크·외부 런타임 없이 순수 HTML/CSS/JS
- **단일 파일 지향**: 페이지별로 가볍게, 네트워크 의존성 없음
- **반응형**: 넓은 화면은 다열, 좁은 화면은 1열로 재배치
- **테마 변수**: `:root`의 CSS 변수(`--ddwu-wine`, `--ddwu-gold` 등)로 색상 재사용
